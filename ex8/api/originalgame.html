ObjectGame documentation¶

In this module you can find definitions for the available methods to help you create new objects for the bonus part of the exercise.

class originalGame.OriginalGame(game)¶
__init__(game)¶
add_original_torpedo(torpedo)¶

This function allows you to add specaily crafted torpedos.

Parameters:torpedo (BaseObject) – The new torpedo to add
add_shape(name, cords, override=False)¶

Adds a shape, and binds it into the screen so we could paint it later on. If this shape was previously defined than it won’t be changed.

Parameters:
  • name (str) – The name of the shape.
  • cords (tuple of tuples of ints) –

    The coordinates of the shape, for example our ship is defined like so ‘((-10,-10),(0,-5),(10,-10),(0,10))’ -

    our first line will go from (-10,-10) towards (0,-5) then the next line to (10,-10) and from there to (0,10) - this is the last point so it will also close the picture.
bind_key(key, func)¶

This method is to allow you to add some functionality of your own, it allows you to bind the provided function to the desired input key.

If there is already a function bound to this key it will do nothing.

Parameters:
  • key (str) – A key to bind.
  • func (function) – The function to bind
