BaseObject documentation¶

class baseObject.BaseObject(canvas, x, y, speedX, speedY, shape, direction=None, rad=1)¶

This class represents our standard object in the game. All of your classes (ship,asteroid,torpedo) should inherit from this class.

__init__(canvas, x, y, speedX, speedY, shape, direction=None, rad=1)¶

To instanciate a class that inherits from BaseObject you must call in the first line of you __init__ function to: super().__init__(<params>) where the parameters are explained below:

Parameters:
  • canvas (Turtle.ScrolledCanvas) – The canvas on which we should draw our object.
  • x (int) – The x coordinate of the object.
  • y (int) – The y coordinate of the object.
  • xSpeed (int) – The current speed on the x direction.
  • ySpeed (int) – The current speed on the y direction.
  • shape (int) – The shape of the object, should be taken from the module objectShapes.
  • direction – The direction (in degrees) we are currently heading.
  • shape – The radius of the object.
